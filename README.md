# About This Midterm Project
Hello. This is my midterm project submission for the Spring 2026 NYU digital humanities course, Intro to Web Development, taught by profesor Jo Suk. 

## Assignment Instructions

The following is the assignment instructions from our class page:

"Midterm

Make a static website using HTML and CSS that presents research on a topic related to your academic work. The website should incorporate design principles learned in class up until the due date. An exceptional website will:

Have a layout reflective of its wireframe
Meet web content accessibility guidelines (WCAG) covered in class (i.e. alt text, color contrast, text size)
Be responsive, adapting to at least 2 screen sizes fluidly
Make use of color, typography, and composition to create a visual style
Include semantic HTML and CSS classes/ids
Feature relevant content
Feature functional internal/external links
Documentation of your process will factor into the assessment of this project, in addition to the resulting website. Documentation will comprise a wireframe, a brief write-up, and a 10-minute presentation. The write-up should be no less than 500 words and cover 1) your goals for the website, 2) 1-2 design choices you made and the reasoning behind them, and 3) 1-2 technical challenges you solved in the process. The presentation should walk the class through your final website and development process. You may rely on your write-up to guide the presentation.

Source code and documentation should be uploaded to its own GitHub repository. The write-up can go in the README.md file. You can link your wireframe as a link online or as an image file in the README.md.

Keep in mind you will be asked to update your midterm project with basic functionality using JavaScript a few weeks following the deadline, so save any ideas that come up during the process about how to make your site more interactive."

## My Project Proposal

For my static site, I will be focusing on my hobby interest in researching controversies related to food. The website will be a personal collection of foods I have learned lore about that I find personally interesting. The website will be called "Food Scandals" and will link to resources about the food conundrums I'm keeping tabs on.
